EBT-Site

EBT-Site is the template for creating a custom website based on EBT-Vue3.

To create your own EBT website:

  1. Login to Github
  2. Clone this repository to your own account
  3. Customize ebt-config.mjs
  4. Commit your changes to Github
  5. Enable Github Pages

That's it!
